說明

本系統所包含的所有開源軟體包均根據其各自的許可證發布。我們感謝這些專案的貢獻者和維護者，他們的工作使得本系統的開發成為可能。

本系統是基於 Django 框架建置。提供進銷存以及物流相關的功能及報表。Django 是一個高級 Python Web 框架，它鼓勵快速開發和乾淨、實用的設計。
